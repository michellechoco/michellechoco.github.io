Experience

Projects

Machine Learning Robotics Simulation

This project was the final deliverable for a machine learning project course. The goal was to develop an agent to navigate a parking lot while classifying the license plates of parked cars. I extracted license plates from cars through HSV thresholding, contour detection, and perspective transform, and constructed a CNN to classify individual characters using TensorFlow.
Skills: OpenCV, TensorFlow, Python, ROS, Linux

Autonomous Robotics Competition

The finale of an Engineering Physics design course. Within a team of 4, we designed and built 4 fully autonomous robots to find, pick up, and deposit cans. Each member of my team built a robot arm with 3 degrees of freedom to reorient cans and to trigger a lever for can drop-off. After battling it out head-to-head with 15 other teams, my team came out 4th place.
Skills: C/C++, Oscilloscope, Soldering

Image Processing Project

I created image processing classes in Java to manipulate images in JPEG and PNG formats, allowing users to green screen, posterize, and recolour images.
Skills: Java, JUnit
